Make Responsive layout easy with CSS

Making website responsive can become headache if you have not planned for it in advance, and the situation becomes even worse when you have to deal with so many screen sizes for desktop, tablet & mobile, and with each device brand having different screen sizes.

Here are the 3 tips to make responsive layout easy :-

  1. Instead of using fixed sizes like pixels, start using variable sizes like rem, em & % for styling, this will ensure the change of your HTML element size with change of screen size, it will solve most of your problems. If you struggle to convert pixel size to rem, use pixel to rem calculator instead. Especially dont give fixed sizes to height and width value, because when you give fixed height to your div container suppose, and when width decreases in mobile view the height remains the same due to fixed height, which leads to overflow of content in the div container, and the best way is to dont give height at all, let the content in the div container take its height, dont set any height if not necessary.
  2. Build mobile layout first instead of building desktop first & then doing the mobile, because mobile layout's are generally simple compare to desktop layout, so it becomes easy to then add complexity like flexbox or grid for desktop layout using media query.
  3. Don't try to create tons of media queries trying to target every device width that exist, with so many phones, tablets & screen sizes its simply not practical, your only going to get confused, instead try to write one breakpoint for mobile with width 700px, that will cover every mobile device somewhere in this range, and if you want to target tablet screen size also, then use around 800-900px accordingly, this way you would be writing less media query by avoiding repetitive code.
